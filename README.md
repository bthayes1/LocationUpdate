# LocationUpdate
### *Brandon Hayes*

## **What is this?** 
This a simple program that is used to show the current location of the user's device on a map. 

## **Why does this exist** 
I wanted to practice with permission requests. 

## **Fun things used:**
* FusedLocationProvider
* Google Maps API
* Permission Requests
